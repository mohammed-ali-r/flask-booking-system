# Flask Booking System

This project is a Flask-based booking system developed as part of my T-Level in Digital Design, Development and Production.

## Features
- User booking functionality
- Flask backend with SQLite database
- HTML/CSS frontend
- Basic form validation

## Technologies Used
- Python (Flask)
- HTML / CSS
- SQLite

## Purpose
This project was built to demonstrate full project lifecycle development, including requirements gathering, development, testing, and deployment concepts.

## Notes
This project is for demonstration and learning purposes.
